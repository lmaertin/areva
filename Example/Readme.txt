1) Run AReva as eclipse application
2) Quit AReva
3) Copy the contents of this folder to workspace/"runtime-EclipseApplication" (Override areva.xml)
4) Start AReva
5) Seletec Perspective "ArevaDefault" and than select konfortsyste:sysarch_120110.arxml as architecture and AutosarExportExplorer.class as parser in XMLSelection-View
6) Select "evaluate" in context menu of the root node of the xml file "MyEvaluations.areva"
7) Press "E" in the tool bar
8) Inspect the results in the perspective "Evaluation"
